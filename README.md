Проект для Яндекс Лицея.

О проекте

Представляет собой собственное расширение для изображений, под названием UTI. С помощью него вы сможете безопасно обмениваться изображениями. Поддерживает ассоциации. Имеет ярлыки в меню пуск.

Состав проекта

- Конвертатор
- Painter (Для UTI и PNG)
- Просмоторщик
- Установщик, обновлятор и удалитель
- Сервер:
-- ТГ бот
-- Сервер, устанавливающий UTI

Установка

!ТРЕБУЮТСЯ ПРАВА АДМИНИСТРАТОРА!

Есть два варианта установки:
1 - Работает в любом случае.
2 - Работает только когда сервер запущен (почти никогда).

1 начинается с 1 шага, а 2 - с 6 шага.

1. Скачайте и распакуйте все файлы из папки UTI/Compiled/Server + Client (source).
2. Запакуйте все файлы из папки Source (unzipped) в архив source.zip и поместите source.zip в папку, которая содаержит распакованные файлы из 1 шага.
(Директория на этом шаге выглядит примерно так:
- bot.exe
- server.exe
- source.zip
-- Inks
-- Program
-- execute.exe
...)
3. Очистите файл token.txt.
4. Получите ключ для ТГ бота (создайте его при необходимости) и введите его в token.txt.
5. Запустите файл start.bat.
6. Скачайте установщик из UTI/Compiled/Installer и запустите его.
7. Разрешите ему вносить изменения на вашем компьютере.
8. Следуйте инструкциям установщика (если вы выполняли пункты 1-5, то можете пропустить шаг с установкой Radmin).
9. Наслаждайтесь UTI!
